# Battle Bounce

**Status:** In Progress

## About
Battle Bounce is a physics-based 2D action fighting game where relies on gun knockback and grappling. The game emphasizes fast-paced movement, strategic bouncing, and competitive player interactions.

## Features Implemented
- Player movement and bouncing mechanics
- Collision detection
- Health system
- Multiple maps
- Player ranking system
- Time limits
- 4 unique characters
- All art assets

## Planned Features
- Enemy AI
- Interactive stage hazards
- Enhanced visual effects and polish
- Music
- SFX
- Character selection screen

## How to Play/Run
- Open the project in Unity 2022.3.4f1 or later
- Open the scene: 'TitleScreen.unity'
- Press Play

## Controls
- Left stick: Control gun direction
- Right stick: Control grapple direction
- Left trigger: Shoot gun
- Right trigger: Shoot grapple
- Right trigger (blue character only): Teleport

## Notes
- This project is a work in progress
- This project is a controller-only game, keyboard and mouse support is not a planned feature
